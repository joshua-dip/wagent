# 카카오 앱 아이콘 등록 가이드

## 📋 **아이콘 요구사항**

### 필수 규격
- **크기**: 512 x 512 픽셀
- **형식**: PNG 또는 JPG
- **용량**: 최대 500KB
- **배경**: 투명 배경 권장 (PNG)
- **비율**: 정사각형 (1:1)

### 디자인 가이드라인
- ✅ 심플하고 명확한 디자인
- ✅ 브랜드 컬러 사용
- ✅ 작은 크기에서도 식별 가능
- ❌ 복잡한 텍스트 지양
- ❌ 너무 많은 디테일 지양

## 🛠 **아이콘 제작 방법**

### 방법 1: Canva 사용 (무료, 쉬움)
1. [Canva](https://www.canva.com/) 접속
2. "사용자 지정 크기" → 512 x 512 px
3. 템플릿 선택 또는 직접 디자인
4. PNG로 다운로드 (투명 배경)

### 방법 2: Figma 사용 (무료, 전문가용)
1. [Figma](https://www.figma.com/) 접속
2. 512 x 512 px 프레임 생성
3. 아이콘 디자인
4. Export → PNG (2x)

### 방법 3: AI 생성 (빠름)
```
프롬프트 예시:
"Create a simple, modern app icon for an English education platform called PAYPERIC. 
Blue and white color scheme. 512x512 pixels. Minimalist design with letter P or book symbol."
```
- [DALL-E](https://openai.com/dall-e-2)
- [Midjourney](https://www.midjourney.com/)
- [Adobe Firefly](https://www.adobe.com/products/firefly.html)

### 방법 4: 온라인 아이콘 제작 도구
- [Hatchful by Shopify](https://www.shopify.com/tools/logo-maker) - 무료
- [LogoMakr](https://logomakr.com/) - 무료
- [Brandmark](https://brandmark.io/) - 유료

## 📤 **카카오 개발자 콘솔에 등록**

### 1. 아이콘 업로드
1. [Kakao Developers](https://developers.kakao.com/) 접속
2. 내 애플리케이션 선택
3. **앱 설정 > 요약 정보**
4. **앱 아이콘** 섹션에서 "등록" 클릭
5. 512x512 PNG/JPG 파일 업로드
6. 저장

### 2. 확인 위치
아이콘이 표시되는 곳:
- ✅ 카카오 로그인 동의 화면
- ✅ 카카오톡 알림 메시지
- ✅ 내 애플리케이션 목록
- ✅ 사용자의 연결된 서비스 목록
